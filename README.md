# SDB Search Database Digital Materials

This repository stores the digital materials submitted with the capstone project
on historical Hong Kong photographs, machine reading, and the cleaned SDB image
collection.

## Packages

- `SDB_image_library_cleaned_313.zip`
  - Cleaned SDB image library with 313 JPG files named `SDB-001` through
    `SDB-313`.
- `SDB_dataset_and_analysis_cleaned_313.zip`
  - Cleaned English coded dataset, image-embedded workbook, cleaned ID mapping,
    and statistical analysis package.
- `SDB_experiment_records.zip`
  - Manual image-reading workbook, 37 selected SDB close-reading images, and
    retained AI experiment output records.

The ZIP packages are tracked with Git LFS.
