# SDB Search Database Digital Materials

This repository stores the digital materials submitted with the capstone project
on historical Hong Kong photographs, machine reading, and the cleaned SDB image
collection.

## Direct Downloads

- [SDB_image_library_cleaned_313.zip](https://github.com/qinhechuan1010-png/SDB-search-database/releases/download/digital-materials-v1/SDB_image_library_cleaned_313.zip)
  - Cleaned SDB image library with 313 JPG files named `SDB-001` through
    `SDB-313`.
- [SDB_dataset_and_analysis_cleaned_313.zip](https://github.com/qinhechuan1010-png/SDB-search-database/releases/download/digital-materials-v1/SDB_dataset_and_analysis_cleaned_313.zip)
  - Cleaned English coded dataset, image-embedded workbook, cleaned ID mapping,
    and statistical analysis package.
- [SDB_experiment_records.zip](https://github.com/qinhechuan1010-png/SDB-search-database/releases/download/digital-materials-v1/SDB_experiment_records.zip)
  - Manual image-reading workbook, 37 selected SDB close-reading images, and
    retained AI experiment output records.

The ZIP files are published as GitHub Release assets so they can be downloaded
directly without using Git LFS.
